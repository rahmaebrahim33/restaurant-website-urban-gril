## 📌 Project Overview

This project showcases a fully interactive restaurant website that simulates a real-world ordering experience.

The goal was to create a visually appealing and user-friendly interface that allows customers to بسهولة:

* Browse the menu
* Add items to cart
* Complete a checkout process

---

## 🎯 Problem

Many restaurant websites lack a smooth and intuitive ordering experience, making it difficult for users to quickly place orders.

---

## 💡 Solution

I designed and developed a complete user flow that includes:

* Interactive menu browsing
* Dynamic cart system
* Checkout form for user details

---

## ✨ Features

### 🧾 Interactive Menu

* Categorized items (e.g., Pizza, Burgers, Drinks)
* Food images, prices, and descriptions
* Add-to-cart functionality

### 🛒 Cart System

* Add / remove items
* Update quantity
* Real-time total price calculation

### 🧍 Checkout Experience

* User details form (Name, Address, Phone)
* Order summary before confirmation
* Clean and simple UI for better usability

### 🎨 UI & UX

* Modern dark-themed design
* Smooth animations and transitions
* Hover effects and micro-interactions
* Fully responsive (mobile, tablet, desktop)
